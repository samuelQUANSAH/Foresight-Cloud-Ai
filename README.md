## Multi-agent orchestration demo

[`claimsense-agentic-triage/`](claimsense-agentic-triage/) contains a working multi-agent
orchestration system: a supervisor agent coordinates Planner, Retrieval, Compliance,
Decision, and Critic agents over a shared state object, with policy-grounded retrieval,
PII/compliance guardrails, confidence-based human-review escalation, and structured
tracing. See its [README](claimsense-agentic-triage/README.md) for setup and a sample
API request.
